# SSH 远程监控桌面小组件

一个基于 Electron + SSH 的轻量桌面监控工具，每隔 5 秒拉取一次远程 Linux 指标并展示在桌面小窗中。

## 支持指标

- CPU: 型号、内核数、主频、使用率、Top5 CPU 进程
- 内存: 总量、使用量、使用率、Top5 内存进程
- 网络: 收发速率、估算使用率、Top5 网络进程（有 `nethogs` 时为流量 Top5，否则回退为连接数 Top5）

## 快速开始

1. 安装依赖

```bash
npm install
```

2. 运行

```bash
npm start
```

3. 在左侧填写 SSH 信息，点击“启动监控”

## 说明

- 监控刷新间隔固定为 5 秒。
- 支持密码或私钥登录（二选一，私钥路径填写本机可读路径）。
- 支持设置：
  - 背景颜色
  - 背景图
  - 背景透明度
  - 字体大小与字体
  - CPU/内存/网络环形图颜色
  - 开机自启动

## 远程机器依赖建议

- 推荐安装 `procps`, `iproute2`（通常默认已有）
- 如要显示网络流量 Top5 进程，建议安装 `nethogs` 并允许无密码 sudo 执行

> 未安装 `nethogs` 时，网络 Top5 将显示为“连接数 Top5 进程”。
